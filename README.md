# CricketArena 🏏

AI-powered gamified cricket prediction platform built with **Python + Streamlit + Google Gemini**.

## Features
- 🏠 **Dashboard** — Live match stats, XP progress, streak tracker, Gemini insights
- 🎯 **Predict** — Lock in match winner, top scorer, and score range predictions
- ⚡ **Fantasy** — Build your XI within a 100-credit budget
- 🏆 **Leaderboard** — Global rankings with streak badges
- ✨ **XP & Badges** — Level progression, badge collection, XP history
- 🤖 **AI Coach** — Chat with Google Gemini for cricket strategy advice

## Live App
👉 [cricketarena.streamlit.app](https://share.streamlit.io)

## Run Locally
```bash
cd streamlit_app
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- **Frontend**: Streamlit
- **AI**: Google Gemini 1.5 Flash
- **Deployment**: Streamlit Community Cloud (free)
